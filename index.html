Features

かざすだけで星がわかる。シンプルで、パワフル。

🌌

AR スカイビュー

端末のセンサーと連動し、かざした方向の星空をリアルタイム表示。直感的な操作で夜空を探索できます。

⭐

9,096 恒星データ

Yale Bright Star Catalogue (BSC5) の全データを内蔵。等級・スペクトル型・距離・B-V色指数まで収録。

✨

88 星座表示

IAU公式の全88星座の線画と日本語/英語名を表示。星空のどこに何座があるか一目でわかります。

🪐

惑星位置計算

水星から土星まで6惑星の位置をケプラー軌道要素からリアルタイムで算出。NASA JPL の精密データに基づきます。

🔍

検索 & ナビゲーション

星名・星座名で検索し、画面上の矢印ガイドで目標の天体まで誘導。目当ての星をすぐに見つけられます。

⏰

時刻シミュレーション

スライダーで日時を自由に変更。昨夜の星空も、来月の星空も、好きな時刻の空を再現できます。

📴

完全オフライン動作

全データ・全計算を端末内で完結。山岳地帯、離島、キャンプ場、飛行機内 — 電波がなくても使えます。

🔴

ナイトモード

画面を赤色系に切り替え、暗順応を妨げません。天体観測のお供に最適です。

Technology

天文学の計算理論をポケットに。

Coordinate Transform

恒星時計算(IAU 1982) → 赤道座標を地平座標に変換 → センサー姿勢行列で端末座標系に投影。天球上の星の位置を、あなたのスマホの画面座標にマッピングします。

Sensor Fusion

加速度センサ + ジャイロセンサ + 磁気センサを相補フィルター(α=0.98)で融合。ジャイロの高速応答と加速度/磁気の安定性を両立し、滑らかなトラッキングを実現。

Gnomonic Projection

3D端末座標から2Dスクリーン座標への変換に心射影(Gnomonic Projection)を採用。広い視野角(30°~120°)でも直線が直線として描画される自然な見え方。

Stellar Color

B-V色指数 → 色温度(Ballesteros 2012) → RGB値(Tanner Helland)で恒星の実際の色を忠実に再現。青白いシリウスから赤いベテルギウスまで。

Kepler Solver

NASA JPL低精度軌道要素とケプラー方程式(Newton-Raphson法)で惑星位置を算出。地心黄道座標への変換まですべて端末内で実行。精度 ±1°。

Flutter + Custom Painter

Flutter の CustomPainter で 2D Canvas に直接描画。Riverpod による状態管理と 60fps の滑らかなレンダリングを両立。
